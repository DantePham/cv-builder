# ⚡ QUICK START - CHẠY SERVER NGAY

## 📝 3 BƯỚC ĐƠN GIẢN:

### 1️⃣ Mở Terminal
```powershell
cd d:\Code\test\cv-builder
```

### 2️⃣ Cài dependencies (chỉ lần đầu)
```powershell
npm install
```

### 3️⃣ Chạy server
```powershell
npm start
```

## 💾 LƯU CV

1. Login
2. Nhập thông tin
3. Click **"💾 Lưu Thông Tin"**
4. Check console (F12) → Thấy "✅ Saved to backend"
